В этом проекте я создаю приложение на Python, с помощью которого можно будет проверять и увеличивать собственную скорость набора текста. Для графического интерфейса используем библиотеку PyQt5, которая для этих целей и предназначена.

При создании проекта я использую принципы объектно-ориентированного программирования.

Мой проект состоит из пяти форм:
-1) Регистрация и вход пользователя.
-2) Главаная(ссылочная) форма, которая открывает доступ к фунционалу приложения.
-3) "Настройки". В ней можно выбирать и редактировать текст
-4) Сама тренировка
-5) Формирование результатов в виде графика
